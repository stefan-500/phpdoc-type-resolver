# PHP Variable Type Resolver

This is a small Java project for infering a PHP variable type from its `@var` PHPDoc annotation.

## Tech Stack

- Java 21
- Gradle
- JUnit 5

## Running tests
To run the tests, execute the following command in the project directory:  

Linux/Mac:
```bash
./gradlew test
```
Windows:
```bash
gradlew test
```
